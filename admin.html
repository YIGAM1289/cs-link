🛠 CS 상담방 관리

상담방 추가·삭제와 휴무 처리를 여기서 관리합니다.
저장하면 고객센터 페이지에 자동 반영됩니다.

🔑 관리자 인증

토큰 만드는 방법 (최초 1회)
  1. 저장소 소유 계정(YIGAM1289)으로 GitHub 로그인
  2. Fine-grained token 만들기 접속
  3. Token name: cs-link-admin, Expiration: 1년
  4. Repository access: Only select repositories → cs-link 선택
  5. Permissions → Repository permissions → Contents: Read and write
  6. Generate token 후 github_pat_… 값을 위에 붙여넣고 토큰 저장
